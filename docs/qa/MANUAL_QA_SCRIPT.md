# Manual QA Script

This script gives a repeatable manual pass for local demos.

## Setup

1. Start MongoDB.
2. Start the backend.
3. Seed demo data.
4. Start the web app.
5. Start the mobile app if mobile QA is in scope.

## Citizen Pass

1. Log in as `demo@ikomek.kz`.
2. Open dashboard.
3. Create a request.
4. Open the request details.
5. Send or view a request message.

## Operator Pass

1. Log in as `operator@ikomek.kz`.
2. Open operator dashboard.
3. Find the citizen request.
4. Update status.

## Admin Pass

1. Log in as `admin@ikomek.kz`.
2. Open admin dashboard.
3. Review analytics.
4. Check users and news management.
