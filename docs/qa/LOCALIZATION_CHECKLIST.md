# Localization Checklist

The platform supports Russian, Kazakh, and English content.

## Shared Checks

- Confirm each primary screen has translated labels.
- Confirm missing translations fall back gracefully.
- Confirm long labels do not overflow buttons, tabs, or cards.
- Confirm date and status labels are understandable in each language.

## Backend Checks

- Categories include localized names.
- News data preserves translated title and content fields when present.
- Request status values can be mapped by each client.

## Web Checks

- Switch language and refresh the page.
- Confirm navigation, forms, validation messages, and empty states update.
- Confirm role-specific pages keep translated labels.

## Mobile Checks

- Switch language in profile or settings.
- Restart the app and confirm the saved language is applied.
- Confirm tab labels and request flow labels fit on small screens.
