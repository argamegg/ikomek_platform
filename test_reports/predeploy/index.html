  • Web: ESLint и production build подтверждают качество React/TypeScript-кода и готовность Vite-сборки; npm audit ищет известные уязвимости JavaScript-пакетов.
  • Mobile: Expo lint, TypeScript и Expo doctor проверяют React Native/Expo проект, конфигурацию SDK и типовую корректность; npm audit проверяет мобильные зависимости.
  • Security: dependency audit ищет CVE/advisories, secret-pattern scan подсвечивает потенциальные секреты и дефолтные ключи, которые нужно проверить перед production.
  • Performance: размер build/export артефактов помогает заранее увидеть слишком тяжелые сборки и риск медленной загрузки.

Test Overview

This report summarizes deployment readiness for the iKOMEK project across the backend API, web frontend, and mobile app. The suite combines functional smoke tests, production builds, TypeScript/ESLint validation, dependency vulnerability audits, static security scanning, and basic artifact-size checks.

The purpose is to catch deployment blockers before release, confirm that critical API contracts still work, verify that dependencies have no known vulnerabilities, ensure production builds are valid, and validate that the mobile project matches Expo SDK expectations.

  • Backend: pytest validates security-sensitive helpers, JWT-protected API contracts, and route structure; pip check and pip-audit verify Python dependency consistency and vulnerabilities.
  • Web: ESLint and production build validate React/TypeScript quality and Vite deployment readiness; npm audit checks JavaScript packages for known advisories.
  • Mobile: Expo lint, TypeScript, and Expo doctor validate the React Native/Expo app, SDK configuration, and type correctness; npm audit checks mobile dependencies.
  • Security: dependency audits search for CVEs/advisories, while the secret-pattern scan highlights possible secrets and default keys that need review before production.
  • Performance: build/export artifact sizes help identify heavy bundles and possible slow-load risks early.
